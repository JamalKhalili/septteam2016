Overview

Main function of this system is to retrieve data from Australia's Bureau of Meteorology website www.bom.gov.au for weather stations around Australia and its Territories (including the Australian Antarctic Territory). The system allows visitors to select weather stations they want to keep track of (their favourites).

Getting Started

This program runs on php 5. In short, once you have installed Apache, enabled php5 and set up your local web server. Place the code folder into the documents folder where your locally hosted files run. Then, in your browser write the name of your local host followed by /code (e.g. for Mac 'localhost/code'). First you must make sure you have installed apache. The following links will give you a set by set guide to installing apache.

Mac http://jason.pureconcepts.net/2014/11/install-apache-php-mysql-mac-os-x-yosemite/

The above link will also explain how to enable php 5. From here, you must place the 'code' folder into /Library/WebServer/Documents. Once you have done so type 'localhost/code' into your web browser of choice and the website will run.

Windows
https://httpd.apache.org/docs/current/platform/windows.html

After installing apache, similar to Mac you must uncomment LoadModule php5_module libexec/apache2/libphp5.so by removing the #. All of this is done in your httpd.conf. Once your have made the necessary changes restart apache (apachectl restart).

This youtube clip shows you how to set up your localhost, to host the website locally.
